My name is Arne Beer. I am currently studying for my bachelor degree of computer science while living and working in Hamburg.

I professionally worked as a web developer for the past three years. During this time I've held roles as backend and frontend developer, which taught me team communication skills in different environments and roles. I worked both in a quite big company and in an one team startup, which taught me to navigate corporate hierarchies as well as working in a small dynamic team without a dedicated project manager. Working in different projects, I became acquainted with many different technologies and paradigms, which gave me a good overview of modern technologies and trends.

I consider best practices such as version control and a properly configured development environment to be a necessity. In my opinion code should be well documented, tested and easy to read.

As an open source enthusiast, I develop software such as command line tools or game development libraries and I do some packaging for Arch Linux's Aur.

In addition to plain programming, I am very interested in infrastructure management in the form of continuous integration and Unix server administration.



In my leasure time I like visiting events like the Global Game Jam or the Chaos Communication Congress to learn about new technologies or programming languages and use the opportunity to test them at the same time. Colaborative programming with my friends is a welcome side effect.

All in all, I just love writing clean and performant software!
